# 変更履歴

## 2026-04-26: プロジェクト基盤、環境変数展開、およびバリデーション・実行機能の実装

- **変更内容**:
  - `clap` による CLI 基盤の実装 (`run` サブコマンド)。
  - `Config` モジュールによる `secrets.json` と `settings.json` の読み込み実装。
  - `EnvManager` モジュールによる `.env` パースと `WITH{{KEY}}` プレースホルダ展開の実装。
  - 開発用設定ディレクトリ `.dev_config` の導入。
  - `Settings` によるセキュリティバリデーションの実装 (`allow_run_path`, `allow_command`)。
  - 外部プロセスの起動および環境変数注入機能の実装。
  - `settings.json` による使用シェルの指定機能の実装。
  - **Nushellの組み込みコマンドとの衝突を避けるため、バイナリ名を `with` に変更。**
  - リリースビルド時の設定ファイル参照先を、OS標準（AppData等）ではなく、Unixライクな `~/.config/with/` に統一。
- **変更理由**: [01\_実装プラン.md](../plans/01_実装プラン.md) の全フェーズを完了させ、APIキー隠蔽・管理CLIツールのコア機能を全て提供するため。また、利用環境（Nushell）との互換性と、環境を問わない設定パスの統一性を確保するため。
- **参照したコーディング計画**: [01\_実装プラン.md](../plans/01_実装プラン.md)
